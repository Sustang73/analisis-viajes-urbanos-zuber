## Descripción del proyecto

Este proyecto analiza datos de viajes urbanos de la empresa ficticia Zuber, con el objetivo de identificar patrones de movilidad, evaluar el comportamiento de las empresas de taxis y analizar si las condiciones climáticas influyen en la duración de los viajes.

## Objetivo

Analizar información de viajes urbanos para responder preguntas de negocio relacionadas con:

- Distribución de viajes por empresa de taxis.
- Zonas con mayor demanda.
- Comportamiento de los viajes hacia el aeropuerto.
- Impacto del clima en la duración de los viajes.
- Validación estadística de hipótesis.

## Herramientas utilizadas

- Python
- SQL
- Pandas
- Matplotlib
- Seaborn
- SciPy
- Jupyter Notebook
